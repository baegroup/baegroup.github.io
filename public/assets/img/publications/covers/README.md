# Publication cover images

Place journal cover files here using the publication id from `public/data/publications.json`.

Naming rule:
- `<publication.id>.webp`
- or `<publication.id>.png`
- or `<publication.id>.jpg`
- or `<publication.id>.jpeg`

Example:
- `2025-am-surface-engineering-li-metal.png`

Fallback order used by the page:
1. `webp`
2. `png`
3. `jpg`
4. `jpeg`

If no matching file exists, the page shows a placeholder card.

# News media folders

Use these folders for News page media:

- `gallery/`
- `videos/`
- `instagram/`
- `notion/` (auto-generated by `npm run news:sync:notion`)

In `public/data/news.json`, each item can include:

- `images`: array of image paths

Supported path styles:

1. Full path with extension
   - `assets/img/news/gallery/event-2026-01.jpg`
2. Base path without extension (auto fallback)
   - `assets/img/news/gallery/event-2026-01`
   - fallback order: `.webp` -> `.png` -> `.jpg` -> `.jpeg`

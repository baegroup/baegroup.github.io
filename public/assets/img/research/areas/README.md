# Research Area Images

Each research card reads images by base name and auto-detects extension in this order:

- `.webp`
- `.png`
- `.jpg`
- `.jpeg`

Default base names from `content/en/research.md`:

- `printing-tech`
- `energy-environment-tech`
- `biomedical-tech`

Path pattern used by the page:

- `public/assets/img/research/areas/<base-name>.<ext>`
